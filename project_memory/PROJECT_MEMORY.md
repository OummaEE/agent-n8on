# Jane AI Agent — Project Memory

## Описание
Локальный AI-ассистент на Windows.
Работает через Ollama (например: qwen2.5-coder:14b).
Управляет системой через инструменты (файлы, браузер, n8n, Gmail и т.д.)

Цель:
универсальный локальный агент, который понимает разговорную речь
и выполняет реальные действия на компьютере.

---

## Основные возможности
- Управление файлами
- Создание документов (docx/pdf/xlsx/pptx)
- Веб-автоматизация (stealth + "как я")
- Telegram управление
- Google Drive / Calendar
- CRM BoostCamp
- n8n workflows
- Голосовой ввод
- Память пользователя
- Multi-step reasoning
